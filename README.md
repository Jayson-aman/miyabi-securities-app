# 📈 グローバル金融AIダッシュボード

世界情勢のニュース・要人発言・軍事動向を AI で分析し、
**FX・先物・株式・仮想通貨・新興テーマ銘柄** の価格方向を多角的に予測する
統合型 Streamlit アプリ。

---

## 🌟 搭載機能（全モード）

### 📊 ダッシュボード
- 全市場（FX/株式指数/コモディティ/仮想通貨）の概況を一画面表示
- 発火中アラートのリアルタイム表示

### 🛢 原油先物予測
- WTI / ブレント / 天然ガス / 金 / 銀 の 1分足〜日足予測
- 世界情勢ニュース sentiment + テクニカル指標 → RandomForest 予測

### 📈 株式ビューア
- 日経平均・TOPIX・NYダウ・S&P500・NASDAQ・個別株
- **銘柄スクリーニング**：リーダー / ラガード / 大変動銘柄を自動抽出
- **1分足予測**：個別株の短期方向

### 💱 FXビューア
- 円高/円安 **ピーク時刻予測**（時刻単位で予測）
- **要人発言フェイント検出**：意図的なはぐらかし発言を解析
- **要人プロフィール**：性格・側近・ブラフ傾向を分析
- **軍事動向モニター**：軍事イベントの円相場影響予測

### 🚀 新興・新発掘銘柄スクリーナー（NEW!）
**11テーマ × 約80銘柄** をカバー：
- 🤖 AI・人工知能
- ⚛️ 量子コンピュータ
- 💎 レアアース・重要鉱物
- 🚀 宇宙・衛星
- 🔋 次世代エネルギー（水素・全固体電池）
- 🧬 バイオテック・遺伝子治療
- 🤖 ロボティクス・自動化
- 🌐 サイバーセキュリティ
- 💊 GLP-1・肥満治療
- 💎 半導体製造装置
- 🌊 核融合・次世代原子力（SMR）

各銘柄について：
- **買い時 / 売り時 判定**（強・中・弱）
- **推奨エントリー価格 / 損切り / 利確目標 / リスクリワード比**
- 全テクニカル根拠（RSI/MACD/MA/BB/出来高）

### ₿ 仮想通貨（NEW!）
- BTC / ETH / SOL / XRP / DOGE / 他 主要12銘柄
- BTC-JPY / ETH-JPY も対応
- 1分足予測 + テクニカル分析

### 🔔 アラート機能（NEW!）
価格・テクニカル条件で自動通知：
- 価格上抜け / 下抜け
- 前日比 +X% / -X%
- RSI 売られすぎ / 買われすぎ
- MACD ゴールデン/デッドクロス

設定は永続化され、履歴も保存されます。

### 📈 バックテストエンジン（NEW!）
5つの戦略を過去データで検証：
- 移動平均クロス（MA20/MA50）
- RSI逆張り
- MACDシグナル
- ボリンジャーバンド逆張り
- ブレイクアウト

評価指標：
- 総リターン / バイ&ホールド対比 / α
- 勝率 / 取引回数 / 最大ドローダウン / シャープレシオ
- 全戦略を一括比較も可能

## 📦 セットアップ

```bash
# 1. 仮想環境を作成
python3 -m venv venv
source venv/bin/activate

# 2. 依存ライブラリをインストール
pip install -r requirements.txt

# 3. 起動
streamlit run app.py
```

ブラウザで自動的に `http://localhost:8501` が開きます。

---

## 📁 ファイル構成

| ファイル | 役割 |
|---|---|
| `app.py` | Streamlit メイン UI（全モード統合） |
| `data_fetcher.py` | Yahoo Finance データ取得（FX/先物/株式/仮想通貨/コモディティ） |
| `news_fetcher.py` | Google News 取得 + sentiment 分析 |
| `predictor.py` | 原油先物 RandomForest 予測 |
| `yen_predictor.py` | 円高/円安ピーク時刻予測 |
| `economic_calendar.py` | 経済指標・要人発言分析 + フェイント検出 |
| `person_profiles.py` | 要人プロフィール（性格・側近・ブラフ） |
| `military_monitor.py` | 軍事動向モニター |
| `stock_screener.py` | 株式スクリーニング・1分足予測 |
| `emerging_stocks.py` | **新興・新発掘銘柄分析（買い時/売り時）** |
| `alert_engine.py` | **アラート設定・通知エンジン** |
| `backtester.py` | **戦略バックテストエンジン** |
